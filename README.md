## Arquitetura aplicada a automações

A estruturação de soluções considera separação entre interação do usuário, controle de execução, processamento automatizado e persistência de dados.

```text
Front-end
    ↓
API RPA
    ↓
Automação RPA
    ↓
API BD
    ↓
Banco de Dados Relacional
```

Nesse modelo:

* o front-end aciona e acompanha processos por contratos de API;
* a automação executa integrações, tratamentos e operações sobre sistemas externos;
* a persistência é centralizada em uma camada específica de acesso a dados;
* execuções e itens processados podem ser rastreados por status, ocorrências e evidências;
* reprocessamentos exigem controle de duplicidade e idempotência.
